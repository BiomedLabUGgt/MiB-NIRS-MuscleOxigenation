Detailed Description

MAX30101 Pulse Oximetry and NIRS Module.

This module provides functions to configure and control the AD MAX30101 optical sensor for pulse oximetry (SpO2), heart rate monitoring, and NIRS (Near-Infrared Spectroscopy) measurements for muscle oxygenation analysis.
